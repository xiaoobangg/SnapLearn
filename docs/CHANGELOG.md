# 版本发布记录

## v1.0.0 (2026-07-03)

### 核心功能

- **拍照学单词**：拍照 → OCR 识别 → 选择单词 → LLM 生成知识点（音标/词性/释义/延伸义/例句/记忆技巧）
- **卡片组学习**：知识点分步学习 → 标记掌握/需再学 → 测试（4 种题型：释义选择/单词选择/搭配/拼写）→ 通关
- **每日打卡**：百词斩式标记（认识/模糊/不认识），连续打卡统计，打卡日历，打卡分享图片
- **随机测验**：错题自动入池，按题型维度追踪复习次数，随机抽题强化记忆
- **AI 助手**：DeepSeek/通义千问双模型，RAG 知识库检索，文档管理 Tool（搜索/创建/编辑文档）
- **语音合成**：DashScope CosyVoice + Coze 工作流双 TTS 引擎，单词级音频缓存复用
- **个人中心**：学习统计、打卡日历、头像昵称、打卡分享图片生成

### 技术栈

uni-app (Vue3+TS) · Spring Boot 3.3 + Java 17 · PostgreSQL 16 + pgvector · Flyway · MyBatis-Plus · Spring AI · Element Plus · Nginx + Docker Compose
