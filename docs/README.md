# 文档目录

```
docs/
├── releases/            ← 版本发布记录
│   └── v1.0.0.md
├── backlog/             ← 需求池 + 已完成方案
│   ├── SPRINT-BACKLOG.md          当前待办
│   ├── DONE-resource-reuse.md     已完成：资源复用
│   └── DONE-document-management.md 已完成：文档管理
└── specs/               ← 技术规格文档
    ├── PROJECT.md       项目架构
    ├── DATABASE.md      数据库设计
    ├── REQUIREMENTS.md  业务需求
    └── api.md           API 文档

### 工作流
backlog 讨论 → 确定方案 → 实施 → 完成 → 发布 → releases 更新
```
